1. React Fundamentals
JSX (JavaScript XML)

JSX is a syntax extension that lets you write HTML-like code in JavaScript. Example: const element = <h1>Hello World</h1>;

Components

Functional Components: Simple JavaScript functions that return JSX
Class Components: ES6 classes with render method (legacy approach)

Props (Properties)

Props pass data from parent to child components, like function parameters. They're read-only.

State

State manages data that changes over time within a component
Using useState hook in functional components
this.state in class components
